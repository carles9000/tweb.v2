TWEB para mod Harbour

La manera mas fácil de diseñar pantallas para mod Harbour en la Web

Descripción

TWeb fue portado para el entorno Génesis Profesional, pero se ha decido extraer toda la libreria para poderla usar independientemente. Las ventajas principales son:

  • Creación de pantallas facilmente usando sintaxis xBase
  • Indepediente de cualquier sistema o entorno de programación
  • Multiplaforma (Window/Linux)
  • La base es Bootstrap, ya que es uno de los frameworks mas populares usados en la web
  • Podeis usar numeros ejemplos que hay en la red por ejemplo en W3schools
  • Todos los fuentes estan disponibles, permitiendo modificar y recompilar en un HRB los cambios y usarlo al momento
  • Posibilidad de usar otros frameworks usando la misma metodologia
Nuestros comandos de siempre xBase (GET,SAY,COMBOBOX,CHECKBOX,...) disponibles ya para nuestros PRG en la Web !
