TWEB for mod Harbour V2

The easiest way to design screens for mod Harbour V2 on the Web

Description

TWeb is a framework that allows you to easily design screens on the web quickly and without having much knowledge of web programming, this is one of its objectives. The other objective was the design of the framework with a wide use of the preprocessor to check the ability of modHarbour to use this type of libraries and preprocessor. The main advantages are:

  • Create screens easily using xBase syntax
  • Independent of any system or programming environment
  • Multiplatform (Windows / Linux)
  • The base is Bootstrap, since it is one of the most popular frameworks used on the web
  • You can use many examples that are in the network for example in W3schools
  • All sources are available, allowing changes to be modified and recompiled in an HRB and used immediately
  • Possibility of using other frameworks using the same methodology
Our usual xBase commands (GET, SAY, COMBOBOX, CHECKBOX, ...) now available for our PRGs on the Web!
